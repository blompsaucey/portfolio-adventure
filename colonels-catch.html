Gamifying App Engagement

Designing a lightweight, on-brand in-app game to drive retention and rewards engagement without a major dev lift, in a price-sensitive market.

Problem

KFC operates in a highly competitive, price-sensitive QSR market. Consumers compare apps. Traditional discounts alone were no longer a compelling hook.

Insight

Gamification offered a way to engage users emotionally. By delivering value through interaction, not price.

This aligned with the business' need to test lightweight engagement mechanics without full-scale development.

Challenge

How do you quickly test the impact of gameplay without extensive dev resources?

The challenge was proving a new engagement mechanic in-market, with third-party tech, limited rewards and no ongoing play loop

Project Overview

Colonel's Catch is an interactive in-app game created for KFC, designed to delight and engage users while promoting brand loyalty. The game features fun mechanics, vibrant visuals, and seamless integration with the KFC app experience.

Key Features

  • Engaging, easy-to-learn game mechanics
  • Custom illustrations and animations
  • Responsive design for all devices
  • Integration with KFC's rewards system

Outcomes

The game increased user engagement and time spent in the app, while providing a memorable, branded experience for KFC customers.
